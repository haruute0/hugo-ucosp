So our project involves making Eclipse soft and fuzzy. So we are trying to hide 90% of Eclipse from the beginner programmer. Currently we are working on trying to hide menu commands and tool bar items in a non-random manner. We are also trying to make class and project creation a bit easier and more consistent.

The Eclipse GUI framework RCP is pretty cool. Seems like a useful thing to know about.
