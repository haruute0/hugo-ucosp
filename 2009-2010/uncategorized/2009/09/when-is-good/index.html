When Is Good?

How should a group of people find a time for a meeting? If they all work for the same organization, their availability might already be in a shared system like Outlook (though that’s by no means guaranteed). Ten years ago, Jon Udell pointed out that online tools were a good lightweight solution to this problem. The MarkUs team is already using WhenIsGood to figure out when they can get together; if your project hasn’t already picked a meeting time, maybe you’d like to give it a try too.
