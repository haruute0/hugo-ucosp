Guest post: an update from the POSIT project

28 Oct 2010, 02:13
POSIT

POSIT aims to create a portable, opensource tool for the Android platform to aid search and rescue efforts by allowing the transmission of data between users and to central servers. Mentor Ralph Morelli provided this update: Things are going very well with the POSIT group. The students have now all identified appropriate projects and each is making steady progress on their work. We have been meeting every Thursday for a brief Skype conference during which we discuss any issues that come up.
