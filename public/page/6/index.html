James Zhao writes: This is the first blog post from the UCOSP Umple Team for Winter 2012. We are working on Umple project, which is a modeling tool and programming language which allows one to perform model-oriented programming. My fellow USCOP colleagues have been working on some interesting projects. Adam Dzialoszynski has been working on SQL statement generation, which allows Umple to generate an SQL database schema. Sonya Adams has been working on adding a new feature to the Umple language which allows a class to be declared as immutable.
